ðŸ“Œ Eulerâ€“Maruyama Formula

The Eulerâ€“Maruyama scheme approximates the SDE

dX = a(X,t) dt + b(X,t) dW

using the update rule:

X[n+1] = X[n] + a(X[n], t[n]) * dt + b(X[n], t[n]) * sqrt(dt) * N(0,1)

ðŸŸ¦ Example 1 â€” Wiener Process (Brownian Motion)


import numpy as np
import matplotlib.pyplot as plt

def wiener_process(T=1.0, N=1000):
    dt = T / N
    dW = np.sqrt(dt) * np.random.randn(N)
    W = np.zeros(N+1)
    W[1:] = np.cumsum(dW)
    return W, dt

W, dt = wiener_process(T=1, N=2000)
plt.plot(W)
plt.title("Wiener Process")
plt.show()

ðŸŸ© Example 2 â€” General SDE Solver


def euler_maruyama(a, b, x0, T=1.0, N=1000):
    dt = T / N
    t = np.linspace(0, T, N+1)
    x = np.zeros(N+1)
    x[0] = x0

    for n in range(N):
        dW = np.sqrt(dt) * np.random.randn()
        x[n+1] = x[n] + a(x[n], t[n]) * dt + b(x[n], t[n]) * dW
    return t, x
