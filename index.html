Stochastic Differential Equation Simulation

Euler–Maruyama Method · Detailed Explanation · Interactive Examples

1. What is the Euler–Maruyama Theorem?

The Euler–Maruyama theorem describes the method used to numerically solve stochastic differential equations (SDEs) of the form:

$$ dX_t = a(X_t,t)\,dt + b(X_t,t)\,dW_t $$

Here, W(t) represents a Wiener process (Brownian motion), a(x,t) is the drift, and b(x,t) is the diffusion. The method provides a discrete approximation for simulating paths of stochastic processes.

2. How it works

The Euler–Maruyama method approximates the SDE solution at discrete time steps:

$$ X_{n+1} = X_n + a(X_n,t_n)\Delta t + b(X_n,t_n)\Delta W_n, \quad \Delta W_n \sim N(0, \Delta t) $$

This formula is analogous to the Euler method for ordinary differential equations but includes the stochastic term.

3. Why it is used

  • Simplicity: Easy to implement for many SDEs.
  • Monte Carlo simulations: Efficiently generate multiple random paths.
  • Educational: Helps understand stochastic processes and their behavior.
  • Baseline method: Serves as a reference for more advanced methods like Milstein.

4. Where it is seen in real life

  • Finance: Simulating stock prices, option pricing (Black–Scholes model), stochastic volatility.
  • Physics: Modeling Brownian motion, particle dynamics, thermal fluctuations.
  • Biology: Neuronal firing, population dynamics with noise, gene expression models.
  • Engineering: Systems with sensor noise, control systems, signal processing.

6. Python Examples

Simulate Wiener process:

import numpy as np
import matplotlib.pyplot as plt

T = 1.0
N = 2000
dt = T/N

dW = np.sqrt(dt) * np.random.randn(N)
W = np.concatenate(([0], np.cumsum(dW)))
plt.plot(W)
plt.title("Wiener Process")
plt.xlabel("Step")
plt.ylabel("W(t)")
plt.grid(True)
plt.show()

7. Conclusion

The Euler–Maruyama theorem provides a fundamental, easy-to-use method to simulate stochastic differential equations. It helps visualize stochastic behavior, estimate expected values, and is widely applied across finance, physics, biology, and engineering.
