1. Euler–Maruyama Method

The Euler–Maruyama update formula for an SDE:

$$ X_{n+1} = X_n + a(X_n, t_n)\Delta t + b(X_n, t_n)\Delta W_n $$

3. Python Version of Wiener Simulation

import numpy as np
import matplotlib.pyplot as plt

T = 1.0
N = 2000
dt = T / N

dW = np.sqrt(dt) * np.random.randn(N)
W = np.concatenate(([0], np.cumsum(dW)))

plt.plot(W)
plt.title("Wiener Process (Brownian Motion)")
plt.xlabel("Step")
plt.ylabel("W(t)")
plt.grid(True)
plt.show()

4. General Euler–Maruyama Solver (Python)

def euler_maruyama(a, b, x0, T=1.0, N=1000):
    import numpy as np
    dt = T / N
    t = np.linspace(0, T, N + 1)
    x = np.zeros(N + 1)
    x[0] = x0

    for n in range(N):
        dW = np.sqrt(dt) * np.random.randn()
        x[n+1] = x[n] + a(x[n], t[n]) * dt + b(x[n], t[n]) * dW

    return t, x
